## 三、安全与质量基线

- **认证**:用户双令牌(HS256+设备 claim,Redis 吊销);后台验证码+错密锁定+IP 白名单+TOTP 二步验证(恢复码);RBAC 权限码;操作全留痕。
- **资金一致性**:忧珠全事务+幂等 biz_key+行锁+死锁重试;对账 daemon 每日巡检(冒烟断言 mismatch=0)。
- **内容风控**:文本三级词库热更新;图片机审三态(mock 验证,云驱动即插);全部图片强制我方存储域名白名单。
- **测试**:单测(含 race)+testcontainers 集成(真 MySQL 并发账务)+十套件端到端冒烟(每次发版与每次 push 双跑);当前全绿。
- **可运维**:dev/prod 双 compose;`.env.prod.example` 全量配置项;demo-reset 一键演示态;外部依赖(S3/机审/LLM/推送/Meili)全部"未配置即优雅关闭",零外网可跑全链路。
